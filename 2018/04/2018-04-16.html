Concept Time: Node.js

Node.js: Node.js® is a JavaScript runtime built on Chrome's V8 JavaScript engine. Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient. Node.js' package ecosystem, npm, is the largest ecosystem of open source libraries in the world.

NPM: npm is the package manager for JavaScript and the world’s largest software registry. Discover packages of reusable code — and assemble them in powerful new ways.

Express: Express is a minimal and flexible Node.js web application framework that provides a robust set of features for web and mobile applications.

Routing: Routing refers to how an application’s endpoints (URIs) respond to client requests.

Why Node.js: Single thread and event driven model make sure speed is fast. Non-block I/O
